# Team Task Manager Project

## 📌 Overview

Team Task Manager is a web-based project management application that helps teams organize, assign, track, and manage tasks efficiently. It provides separate dashboards for administrators and team members, enabling smooth collaboration and real-time task monitoring.

## 🚀 Features

### 👨‍💼 Admin Panel

* Create, update, and delete tasks
* Assign tasks to team members
* Monitor task progress
* Manage team members
* View project statistics and reports

### 👥 User Panel

* View assigned tasks
* Update task status (Pending, In Progress, Completed)
* Receive task notifications
* Track personal productivity
* Collaborate with team members

### 📊 Dashboard

* Real-time task tracking
* Progress analytics
* Task completion statistics
* Team performance overview

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Team-Task-Manager-Project.git
```

2. Navigate to project directory

```bash
cd Team-Task-Manager-Project
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```

5. Start backend server

```bash
npm start
```
